# BogleHub — Indirect & Global Competitor Map

> Research Agent 2/6 | Date: 2026-05-14
> Scope: Global tools that serve Hispanic users in English OR could enter the Spanish passive-investing market.

---

## 1. Global ETF Portfolio Analyzers

### 1.1 Simply Wall St

| Attribute | Detail |
|---|---|
| Pricing | Free (5 stocks/month) → $10–25/mo Premium |
| User base | ~4M users, English-first, partial ES translation |
| Core USP | Visual "snowflake" valuation chart, narrative health summaries per stock |
| Key features | Fundamentals analysis, fair value estimates, portfolio health score, news digests |
| Weakness for ES users | Stock-heavy, ETF analysis is shallow; no Bogleheads philosophy; no UCITS/EU ETF focus; Spanish localization is machine-translated and inconsistent |
| **Copy** | The "health narrative" concept — one sentence explaining a metric in plain language |
| **Do NOT copy** | Gamified snowflake chart — visually memorable but confusing for passive investors who don't care about moat/growth/value scoring |

### 1.2 Morningstar Portfolio Manager

| Attribute | Detail |
|---|---|
| Pricing | Free tier (limited holdings, basic X-ray) → Morningstar Investor ~$199/yr |
| ARR estimate | ~$250M (investor subscription segment, public filings) |
| Core USP | Fund ratings (star system), Portfolio X-Ray (overlap analysis), analyst reports |
| Key features | Asset allocation X-Ray, fee analysis, style box, risk-adjusted returns |
| Weakness for ES users | No Spanish UI; EU ETF coverage is good but cluttered; star ratings misused by passive investors; paywalled data that EU platforms provide free (KIID) |
| **Copy** | Portfolio X-Ray overlap visualization — essential for lazy-portfolio ETF holders checking for stock duplication |
| **Do NOT copy** | Star rating system — actively encourages fund-chasing behavior, antithetical to Bogleheads philosophy |

### 1.3 Yahoo Finance Premium

| Attribute | Detail |
|---|---|
| Pricing | Free → Yahoo Finance Plus ~$34.99/mo or ~$249/yr |
| Core USP | Data breadth, watchlists, earnings estimates, community |
| Key features | Portfolio tracking, alerts, screeners, analyst price targets |
| Weakness for ES users | No Spanish UI; ETF data gaps for EU-listed funds; Premium pricing is US-centric value proposition (earnings/analyst data irrelevant for index investors) |
| **Copy** | Real-time alert system for portfolio rebalancing triggers (threshold-based) |
| **Do NOT copy** | Analyst price targets on ETFs — meaningless noise for passive investors |

### 1.4 StockAnalysis.com

| Attribute | Detail |
|---|---|
| Pricing | Free with ads → Pro ~$79/yr |
| Core USP | Clean, fast ETF screener with expense ratio filters; beloved by Reddit FIRE community |
| Key features | ETF holdings breakdown, performance charts, sector exposure, dividend history |
| Weakness for ES users | No Spanish UI; US-listed ETFs only (no UCITS); no portfolio builder |
| **Copy** | Clean UX for ETF data pages — fast-loading, no clutter, instant key metrics above the fold |
| **Do NOT copy** | Ad-supported free tier — degrades trust for financial tools |

### 1.5 Finbox

| Attribute | Detail |
|---|---|
| Pricing | Free limited → Pro $19–49/mo |
| Core USP | Intrinsic value models, DCF calculators, financial data API |
| Key features | Fair value estimates, valuation models, comparable analysis, screeners |
| Weakness for ES users | Stock/fundamental analysis focus; ETFs are afterthoughts; no Spanish UI; not relevant for passive investors |
| **Copy** | Data export functionality (CSV/API) — power users demand it |
| **Do NOT copy** | DCF valuation framing — irrelevant and counterproductive for index fund believers |

### 1.6 JustETF

| Attribute | Detail |
|---|---|
| Pricing | Free → Premium €199/yr |
| Core USP | European ETF database, UCITS-focused, portfolio tracker, German/multi-language |
| Key features | ETF screener (100+ filters), savings plan simulator, portfolio tracker, ES locale available |
| Weakness for ES users | Spanish translation exists but feels secondary; community is Germanic; no AI features; UI is dated (table-heavy); no personalized advice layer |
| **Copy** | Savings plan simulator (monthly DCA projector) — critical for European retail investors who invest via brokers' savings plans; UCITS-first ETF database |
| **Do NOT copy** | Table-dense UI — overwhelming for new investors who are the key growth segment in Spain |

## 2. Portfolio Backtesting Tools

### 2.1 Portfolio Visualizer

| Attribute | Detail |
|---|---|
| Pricing | Free (basic) → Pro $29–59/mo |
| Core USP | Gold standard for backtesting; Monte Carlo simulation; factor regression (Fama-French) |
| Killer features | Backtest Portfolio (historical returns, CAGR, max drawdown, Sharpe), Monte Carlo retirement projections, factor analysis, efficient frontier |
| Weakness for ES users | US data only (no EU ETFs); English-only; charts require financial literacy to interpret; no mobile version |
| **Copy** | Monte Carlo retirement projection UI — visualizing probability cones for retirement outcomes is highly compelling; efficient frontier chart for portfolio construction |
| **Do NOT copy** | Factor regression tool for retail users — overwhelming complexity that drives away 90% of the target audience |

### 2.2 PortfolioCharts

| Attribute | Detail |
|---|---|
| Pricing | Free |
| Core USP | Beautiful visualizations of portfolio risk/return across all historical periods; "heat maps" of calendar returns |
| Killer features | Heat map calendar returns, Ulcer Index, Perpetual Withdrawal Rate, portfolio comparison |
| Weakness for ES users | US/global asset class data; no EU ETF tickers; English-only; no user accounts |
| **Copy** | Calendar heat map returns visualization — instantly intuitive for showing volatility to new investors |
| **Do NOT copy** | No accounts/portfolio builder — limits retention and personalization |

### 2.3 Backtest Bay / testfol.io

| Attribute | Detail |
|---|---|
| Pricing | Free (testfol.io) → Backtest Bay has paid tiers |
| Core USP | Fast, simple backtesting with custom asset allocation; testfol.io is notably clean and fast |
| Key features | Asset allocation backtesting, rebalancing simulation, drawdown charts |
| Weakness for ES users | Limited EU ETF coverage; no Spanish UI; testfol.io lacks depth beyond basic backtest |
| **Copy** | Speed and simplicity of testfol.io — a backtest in under 30 seconds with zero friction onboarding |
| **Do NOT copy** | Paywall on backtest history at Backtest Bay — core backtesting should be freemium-accessible |

## Top 5 Features to Steal

| # | Feature | Source | Why It Wins |
|---|---|---|---|
| 1 | **Portfolio X-Ray (overlap analysis)** | Morningstar | Passive investors with 2-3 ETFs constantly worry about duplication; instant visualization of underlying stock overlap is highly shareable and sticky |
| 2 | **DCA / Savings Plan Simulator** | JustETF | European retail investors invest monthly via savings plans; a "what if I invest €200/month for 20 years" simulator with real UCITS ETFs is the single most viral tool for the Spanish market |
| 3 | **Monte Carlo retirement projection** | Portfolio Visualizer | "Will I have enough to retire?" visualized as a probability cone is emotionally resonant and drives premium conversion; no Spanish tool offers this for EU data |
| 4 | **Natural language ETF search** | Magnifi | "Find me a global equity ETF with TER under 0.20% available on MyInvestor" — zero-friction discovery that replaces 30 minutes of JustETF filter work |
| 5 | **Calendar heat map returns** | PortfolioCharts | Showing annual returns as a color grid makes volatility visceral and educational; converts passive reading into active portfolio reflection |

## Top 3 Anti-Patterns to Avoid

| # | Anti-Pattern | Who Does It | Why It Fails for BogleHub |
|---|---|---|---|
| 1 | **Star ratings / analyst price targets on ETFs** | Morningstar, Yahoo Finance | Encourages performance-chasing and fund-switching — the behavior Bogleheads explicitly preach against. Showing a "4-star" ETF vs "3-star" ETF trains users to think like active investors. Credibility destruction. |
| 2 | **Table-dense UI as default view** | JustETF, Finbox | Power users love tables; new investors (the growth market in Spain) bounce immediately. A table of 47 ETF metrics is the fastest way to lose the person who just listened to their first investing podcast and wants to "just start." Lead with narrative, offer tables as drill-down. |
| 3 | **Paywall on core backtesting** | Backtest Bay, Morningstar | The backtest is the product demo — it converts skeptics into believers. Gating it teaches users to go to Portfolio Visualizer or Curvo instead. Free backtesting with UCITS data is a moat-builder; upsell on depth (longer history, more portfolios, PDF export, Monte Carlo runs). |
